Services

Conception d'interface (UI Design)

Le UI design consite à concevoir l'apparence visuelle de votre site: couleurs, typographies, icônes, boutons, mise en page... tout y est! L'objectif est de créer une interface à la fois esthétique, intuitive à agréable à utiliser.

Développement Front-end

Le front-end transforme les maquettes en page web interactive grâce à HTML, CSS, et JavaScript. C'est la partie visible par l'utilisateur, là où chaque clic, animation ou effet visuel prend vie.

Developpement Back-end

Le back-end, c'est la partie invisible qui gère les bases de données, les serveurs et la logique métier. Il fait fonctionner l'application en arrière plan, en assurant la sécurité, le stockage et le bon traitement des données.

Phase de test (Testing)

Chaque fonctionnalité est testée pour s'assurer qu'elle fonctionne parfaitement, sur tous les appareils et navigateurs. Le testing permet de corriger les bugs, d'optipmiser les performances et de garantir une expérience utilisateur fluide et fiable.

Mes compétences

HTML

HTML est le langage de base qui structure les pages web. Il définit les titres, paragraphes, images et liens visibles à l’écran.

CSS

Le CSS apporte du style et de la vie aux pages : couleurs, animations, mise en page responsive et design visuel.

JavaScript

JavaScript rend les sites interactifs et dynamiques. Il gère les animations, les événements et les comportements en temps réel.

WordPress

WordPress est un CMS puissant et populaire pour créer des sites web facilement, avec ou sans code, grâce à ses nombreux thèmes et plugins.

Maintenance informatique

Assistance, installation de logiciels, nettoyage système, optimisation des performances : je m'assure du bon fonctionnement des postes informatiques.

PHP&SQL
